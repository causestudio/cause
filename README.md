Public issue tracking, documentation, and release notes for cause.

cause is an AI-native developer tool for debugging, understanding, verifying, and reasoning about AI-generated code.

## Repository scope

This public repository is used for:

- Bug reports
- Feature requests
- Public roadmap discussion
- Release notes
- Product documentation

The implementation source code is maintained in private repositories under the cause.studio organization.
